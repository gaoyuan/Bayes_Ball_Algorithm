Bayes Ball Algorithm

  • Click in the open space to add a state
  • Drag between states to add an edge
  • Hold 'm' and drag a state to M(ove) state
  • Click on a state or an edge to select it
  • When a state is selected:
    • T(est) marks it as test state
    • Delete removes the state
  • When an edge is selected:
    • R(everse) change direction
    • Delete removes the edge
